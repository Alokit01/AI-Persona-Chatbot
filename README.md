# 🤖 AI Persona Chatbot

A simple AI chatbot built using **Streamlit**, **LangChain**, and **Mistral AI**. The chatbot allows users to interact with different AI personalities through a clean and interactive interface.

## Features

- 🤬 Angry AI
- 😂 Funny AI
- 🌧️ Sad AI
- 📚 Teacher AI
- 💻 Coder AI
- Modern Streamlit UI
- Chat history support
- Switch between different personas

## Tech Stack

- Python
- Streamlit
- LangChain
- Mistral AI
- python-dotenv

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/Alokit01/AI-Persona-Chatbot.git
```

Go to the project directory

```bash
cd AI-Persona-Chatbot
```

Install the required packages

```bash
pip install -r requirements.txt
```

Create a `.env` file and add your Mistral API key

```env
MISTRAL_API_KEY=your_api_key_here
```

Run the application

```bash
streamlit run app1.py
```
